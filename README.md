# CRUD SIMPLES

## Atributos

- CPF: cpf
- nome: name
- dt_nasc: dt_birth
- salario: salary
- [PERSON]
- name: Text
- cpf : varchar(14)
- dt_birth : Date
- salary : Number

## Tecnologia

- HTML5
- Javascript
- Vue.js
- Database: Firebase nosql

## Pages:

- Input (nome, cpf, dt_nasc, salario)
- Listview()
- add()
- Edit()
- Remove()
- Procurar()
- - Filtro(com '< maior q' '> menor q', '== igual q);
